# Shottr

- 이미지 일부를 크롭한 상태에서
  - `A`를 누르면 영역을 보기좋게 잘라준다.  
  - `Enter`를 누르면 영역을 보기좋게 잘라준다.
  - `C`를 누르면 영역에서 평균적으로 쓰인 색깔을 복사한다. 

- `SHIFT` + `TAB`
  - 해당 포인터에서 가리키는 색을 카피한다.
